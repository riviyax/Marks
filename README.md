
# 🎓 MMU Marks System

A simple **MMU Students Database** built using the **MERN stack**.


> Proudly made by **GGRiviya**

---

## 📌 Tech Stack

- **Frontend**: ReactJS, TailwindCSS  
- **Backend**: ExpressJS, MongoDB  

---

## 🚀 Getting Started

### 🖥️ Frontend Setup

To deploy the frontend locally:

```bash
cd client
npm install
npm run dev
```

### 🛠️ Backend Setup
To deploy the backend locally:

```bash
  cd server
  npm install
  npm start
```
